## 关于软件与文件格式
为什么不是pdf或者xmind格式？

我想说一下我这个脑图是支持编辑的，然后marginnote3也是目前我见过的最强大的脑图编辑器。大家用这个格式，下载一下这个软件就可以之间打开，编辑。
